# SOP 标准操作流程
standard operating procedure

## 目的

为确保并明确相关操作要求与执行标准，特制定本文件。

全体 APOCFLY 管制员均应遵循本文件所列之标准操作流程。如在具体情境中遇到与本文件不一致或存在疑问的情况，可在确保安全与协调的前提下，经评估后临时调整执行方式。

若本文件与所引用教程存在差异，应以本文件规定为准。
