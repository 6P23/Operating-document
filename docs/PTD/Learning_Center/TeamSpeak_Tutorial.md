## 正式安装

![image001](./TeamSpeak_assets/image001.png)

*注：本教程以TeamSpeak3-Client-win64-3.6.1为素材制作，目前可能版本过久，但内容近似。*

1. 打开"TeamSpeak3-Client-win64-3.6.1.exe"

	![image002](./TeamSpeak_assets/image002.png)
	
	![image003](./TeamSpeak_assets/image003.png)

2. 划到最底下，同意协议

	![image006](./TeamSpeak_assets/image006.png)

3. 选择安装用户，默认即可

	![image007](./TeamSpeak_assets/image007.png)

4. 选择安装目录，默认即可

	![image008](./TeamSpeak_assets/image008.png)
	
	![image009](./TeamSpeak_assets/image009.png)
	
	![image010](./TeamSpeak_assets/image010.png)
	
	![image011](./TeamSpeak_assets/image011.png)

5. 完成安装，弹出以下界面并关闭[image012](./TeamSpeak_assets/image012.png)

	![image013](./TeamSpeak_assets/image013.png)
